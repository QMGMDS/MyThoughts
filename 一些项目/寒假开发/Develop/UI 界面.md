# 标题界面

![[标题界面 1 1.png]]

需求：
- 基本：
	- 背景图片一张
		参考：找不到合适的！！我预想中可以放个灰暗色调的城市图
- 补充：
	背景图片的主色调也可以是黑色不用怕和交互按钮颜色相撞看不清，因为字体颜色可以改。标题界面的布局大抵是这样，左中放置游戏标题选择按钮，右上放置游戏名，左下放置开发队伍
- 拓展：
	游戏标题字和选择按钮都可以是艺术字体
	鼠标悬停按钮的高亮显示，这里给出两种方案：
	- 每个游戏按钮可以有两张图片，一张是未选中状态下的，一张是选中状态下的
	- 一张高亮显示背景图片，该按钮被选中状态下显示该图片
		例如：![[Pasted image 20260202142626.png]]
	- 背景图片的上层，界面布局的右下角，可以放一台不断弹出红点消息的手机屏幕显示动画
