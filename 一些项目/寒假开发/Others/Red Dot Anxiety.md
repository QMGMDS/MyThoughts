# Red Dot Anxiety - 红点焦虑

手机桌面上，应用软件的红点是否让你深感烦躁？

聊天工具里，各处信息的红点是否让你头晕目眩？

其他界面上，突然的大量红点是否让你手足无措？

你试图关掉红点，可却因此错过了大大小小的许多事情；红点的数量太多消除困难，一个个点击使得你烦躁又无奈；未知的红点代表的究竟是家人朋友的关心，还是漫无止境的工作任务？

在这个游戏里，我们成为不同的人群，尝试更快地消除屏幕上所有令人烦躁、不安、难受的小红点，同时又保证不错过重要的信息，在信息的潮流里，为内心的平静创造一方净土。

## 主要玩法

玩家需要扮演三位主角，帮助他们消除屏幕上所有的通知和红点，完成所有必须做的事情并选择性跳过那些无关紧要的小事。

不同重要类型的事件会有不同的提示方法，玩家可以据此选择不同的处理方式。

每个主角都拥有生命值、疲惫值和焦虑值，不同主角的的属性和系数都不同，需要玩家调整策略应对。

## 具体故事

### 陈默

- 男
- 20岁

- 发展前景迷茫的大学生

- 前路未知，在准备考研和选择就业之间犹豫不决

未知的未来使他害怕。在不同社交圈子里的交流中，寻找自己的前路。



### 林晓

- 女
- 27岁

- 一线城市互联网公司运营，身处职业上升期

- 渴望认可，害怕落后，红点代表着未完成的工作和可能错过的机会


是迎合紧张的工作以获得晋升，还是冒险寻找生活？工作与生活的天平需要你来调整。

### 吴建国

- 男
- 40岁
- 公司中层管理，是儿子、是丈夫、是父亲、也是团队领导。
- 他的时间被自己不同的身份撕成碎片，每一个红点都是一个需要他支撑的世界。

如何将有限的时间分配给不同的“红点”，平衡自己的角色关系，是他的负重前行。
