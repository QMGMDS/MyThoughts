| 功能分类        | 功能说明                           | Windows                  | macOS                 |
| ----------- | ------------------------------ | ------------------------ | --------------------- |
| **项目管理**    | 开关 Project 视图                  | `Alt + 1`                | `Cmd + 1`             |
|             | 叫出 Rider 设置                    | `Ctrl + Alt + S`         | `Cmd + ,`             |
| **编辑操作**    | 复制当前行                          | `Ctrl + D`               | `Cmd + D`             |
|             | 删除当前行                          | `Ctrl + Y`               | `Cmd + Backspace`     |
|             | 上下移动选中行 (可进 `{ }`)             | `Shift + Alt + ↑/↓`      | `Shift + Opt + ↑/↓`   |
|             | 上下移动选中行 (不可进 `{ }`)            | `Shift + Ctrl + ↑/↓`     | `Shift + Cmd + ↑/↓`   |
|             | 在当前行下方新建行                      | `Shift + Enter`          | `Shift + Enter`       |
|             | 在当前行上方新建行                      | `Ctrl + Alt + Enter`     | `Cmd + Opt + Enter`   |
|             | 复制多行 / 列选择                     | `Alt + 左键拖曳`             | `Opt + 左键拖曳`          |
|             | 删除到单词末尾                        | `Ctrl + Delete`          | `Opt + Delete` (常见补充) |
| **导航**      | 跳到定义                           | `Ctrl + 左键`              | `Cmd + 左键`            |
|             | 查找所有引用                         | `Alt + F7`               | `Opt + F7`            |
|             | 移动光标到上一个/下一个词                  | `Ctrl + ←/→`             | `Opt + ←/→`           |
|             | 移动光标到行首/行尾                     | `Home / End`             | `Cmd + ←/→`           |
| **代码生成与重构** | Generate (构造/属性等)              | `Alt + Insert`           | `Cmd + N`             |
|             | 重命名                            | `Shift + F6`             | `Shift + F6`          |
|             | Surround with (try/catch, if等) | `Ctrl + Alt + T`         | `Cmd + Opt + T`       |
|             | Refactor This (重构菜单)           | `Ctrl + Alt + Shift + T` | `^Ctrl + T`           |
|             | 调出 Context Action (快速修复/建议)    | `Alt + Enter`            | `Opt + Enter`         |
| **代码整理**    | 重新格式化代码                        | `Ctrl + Alt + L`         | `Cmd + Opt + L`       |
|             | 优化 / 清除未使用的 using              | `Ctrl + Alt + O`         | `^Ctrl + Opt + O`     |
| **运行与调试**   | 生成解决方案 (Build)                 | `Ctrl + F9`              | `Cmd + F9`            |
|             | 运行                             | `Alt + Shift + F10`      | `^Ctrl + Opt + R`     |
|             | 调试                             | `Alt + Shift + F9`       | `^Ctrl + Opt + D`     |
|             | 停止                             | `Ctrl + F2`              | `Cmd + F2`            |
| **IDE 视图**  | IDE 放大                         | `Alt + Shift + =`        | `^Ctrl + Opt + =`     |
|             | IDE 缩小                         | `Alt + Shift + -`        | `^Ctrl + Opt + -`     |
|             | IDE 重置默认大小                     | `Alt + Shift + 0`        | `^Ctrl + Opt + 0`     |
|             | 全局搜索 (搜一切)                     | `双击 Shift`               | `双击 Shift`            |
