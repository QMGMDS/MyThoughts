# AI Navigation 是什么？
AI Navigation 指 Unity 中已经帮我们实现好的 **导航寻路系统**
该系统是由一个窗口和三个组件所实现的，它们分别是：
1. **Navigation 窗口**
2. **NavMesh Agent组件**
3. OffMesh Link 组件
4. NavMesh Obstacle 组件
总而言之，学习 AI Navigation 就是去学习组成系统的这些东西如何使用
这些窗口和组件的相关说明放在思维导图之中

# 导航寻路系统有什么用？
该系统就做了一件事：
让游戏中的角色能够从一个起点准确的到达另一个终点，并且能够自动避开两个点之间的障碍物选择寻路规则内认为的最近的路径进行前往
导航寻路系统的实现本质就是 **A\*寻路算法** ，相关的说明放在文章末尾的拓展（[[#拓展 - A *寻路|这里跳转]]）

# 寻路系统的工作流程是什么？
要实现这样的一个情景：
游戏中的一个角色要从地图的左下角走到地图的右上角并能自己绕开之间的障碍物

Unity 的寻路系统是这样逐步实现的：
1. 在场景中搭建好各种物体作为地图的样貌（此时的地图只能看不能走）
2. 为该地图设置其在寻路系统中的基本参数
3. 初始化地图（**烘焙 (Bake)**）
4. 给角色一个地图器，之后该角色才会按照地图信息内容行走

利用 Navigation 窗口来设置地图在寻路系统中的基本参数信息
给角色挂载 NavMesh Agent 组件让其按照地图内容行走
