
# 为什么拿手机记笔记

用来记录突发奇想的东西 主意
或者想写什么东西但是没电脑就来这里
所以说电脑还是很重要 或者说电脑来写这种东西真的很舒服
不管是 Markdown 语法在编辑文本中的使用还是想灵活高效的书写笔记 手机写的都太蹩脚了吧()
但是手机也有一个电脑比不上的优点
那就是便携性
用来写突发奇想的东西或者待办再合适不过了
俗话说 不要拿自己的短处和别人的长处比较
既然如此的话 手机只需要记录琐碎的笔记就好了
格式、语法、美观这些东西怎么样都好
只要捕捉到一瞬即逝的想法就是大成功
