
# 第一章：集合

- 集合概念
	- 集合中含有的对象叫做**元素**或**成员** 元素具有**无次序性**和**互异性**
	- 集合中元素个数称为**基数** 集合A基数用 $|A|$ 或 $card(A)$ 表示
	- 基数有限的集合称为**有限集合** 基数无限的集合称为**无限集合**

- 集合表示
	- 枚举法
	- 描述法

- 集合关系
	- 子集
		集合A中元素在集合B中都有 则A是B的子集 记A⊆B
		上述条件下 集合 A≠B 则A是B的真子集 记A⊂B
	- 幂集
		集合A的所有不同子集为元素组成的集合称为集合A的幂集 记P(A)
	- 补集
		不属于集合A但属于全集U中元素组成的集合称为集合A的补集 记~A或Ā

- 集合运算

- 容斥原理

# 第二章：关系

- 序偶
	两个元素按次序排列成的二元组 记为 <x,y>
	前一个为第一元素 后一个为第二元素

- 笛卡尔积
	以集合A中元素为第一元素 集合B中元素为第二元素组成的序偶的集合
	$A×B=${$<x,y>|x∈A,y∈B$}

- 其他关系
	对于任意集合A
	- 空关系
		空集∅
	- 全域关系(包含恒等关系)
		{$<x,y>|x∈A,y∈A$}
	- 恒等关系
		{$<x,x>|x∈A$}

- 关系表示方法
	- 集合法
	- 关系图
	- 关系矩阵(第一元素为行 第二元素为列)

- 关系的性质
	对于集合A上关系R
	- 自反性与反自反性
		任意元素x∈A都有<x,x>∈R
		称关系R有自反性
		任意元素x∈A都有<x,x>∉R
		称关系R有反自反性
	- 对称性与反对称性
		对任意元素x,y∈A
		有<x,y>∈R则有<y,x>∈R
		称关系R有对称性
		仅当x=y才有<x,y>∈R且<y,x>∈R
		称关系R有反对称性
	- 传递性

- 关系运算
	- 基本运算
		关系是序偶组成的集合 所以适用于集合的运算也适用于关系
		并 交 差 补 对称差
	- 特殊运算
		- 复合运算
		- 逆运算
		- 幂运算
		- 闭包运算
			添加最少数量的序偶元素让关系具有对应性质的运算称为闭包运算
			- 自反闭包r(R)
			- 对称闭包s(R)
			- 传递闭包t(R)
			对称自反闭包 先进行自反闭包运算再进行对称闭包运算 记sr(R) 像这样以此类推一堆复合运算
	- 运算封闭性
		关系在经过基本运算或特殊运算后其性质不变

- 特殊关系
	- 等价关系
		具有自反 对称 传递三种性质的关系称为等价关系
	- 相容关系
		具有自反 对称两种性质的关系称为相容关系
	- 偏序关系
		具有自反 反对称 传递三种性质的关系称为偏序关系

- 习题

# 第三章：函数

- 函数概念
	函数是描述两个集合中元素联系的关系 又称为映射
	设f是集合A到B的关系
	对每个x∈A 都存在唯一的y∈B
	使得<x,y>∈f
	一个x只能对应一个y
	一个y可以对应多个x
	集合A中所有元素都要与B中元素有关系
	集合B中允许有元素与A中元素无关系

- 特殊函数
	设f是集合A到B的函数
	- 单射函数
		对A中任意两个元素x和y
		如果x≠y时 都有f(x)≠f(y)
		称f是集合A到B的单射函数
	- 满射函数
		f的值域恰好是集合B
		即f(A)=B

- 函数运算
	函数作为特殊的关系
	- 复合运算
	- 逆运算
