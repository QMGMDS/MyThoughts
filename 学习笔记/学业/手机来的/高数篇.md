# 第八章

- 点到平面距离公式：
	若平面的一般式方程为：
	$Ax + By + Cz + D = 0$
	空间中一点 $P(x_0, y_0, z_0)$ 到该平面的距离 $d$ 为：
	$d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}}$

- 两平面距离公式：
	对于平面
	$Ax+By+Cz+D_1=0$ 和 $Ax+By+Cz+D_2=0$
	它们之间的距离为：
	$d = \frac{|D_1 - D_2|}{\sqrt{A^2+B^2+C^2}}$

- 常见二次曲面的标准形式：
	- 双曲面类
		- 单叶双曲面
			$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$
		- ​双叶双曲面
			$\frac{x^2}{a^2} - \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$
	
	- 抛物面类
		- 椭圆抛物面
			$\frac{x^2}{a^2} + \frac{y^2}{b^2} = z$
		- 双曲抛物面
			$\frac{x^2}{a^2} - \frac{y^2}{b^2} = z$
	
	- 椭球面类
		- 椭球面
			$\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$
		- 球面
			$x^2+y^2+z^2=R^2$
	
	- 锥面与柱面类
		- 圆锥面
			$z^2 = x^2 + y^2$
			$x^2 = y^2 + z^2$
			$y^2 = x^2 + z^2$
		- 椭圆柱面
			$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$
		- 双曲柱面
			$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$
		- 抛物柱面
			$y^2 = 2px$
