# 函数求极限

> [!important] 统一步骤：
第一步：先看类型
第二步：使用无穷小替换（优先）+洛必达法则

> [!summary] 知识储备-无穷小替换：
$$x\sim \sin x\sim \arcsin x\sim \tan x\sim \arctan x\sim e^x-1\sim \ln(1+x)$$
$$1-\cos x\sim \frac{1}{2}x^2$$
$$(1+x)^k-1\sim kx$$


1. $\frac{0}{0}$ 型
> [!example] 例题#1
> 方法一：无穷小替换
$\lim_{ x \to 0 } \frac{x-\sin x}{x^3}$
$=\lim_{ x \to 0 } \frac{1-\cos x}{3x^2}$
$=\lim_{ x \to 0 } \frac{\frac{1}{2}x^2}{3x^2}$
$=\frac{1}{6}$
> 方法二：洛必达
$\lim_{ x \to 0 } \frac{x-\sin x}{x^3}$
$=\lim_{ x \to 0 } \frac{1-\cos x}{3x^2}$
$=\lim_{ x \to 0 } \frac{\sin x}{6x}$
$=\frac{1}{6}$

> [!example] 例题#2
$\lim_{ x \to 0 } \frac{e^x-\sin x-1}{1-\sqrt{ 1-x^2 }}$
$=\lim_{ x \to 0 } \frac{e^x-\sin x-1}{-[(1-x^2)^\frac{1}{2}-1]}$
$=\lim_{ x \to 0 } \frac{e^x-\sin x-1}{\frac{1}{2}x^2}$
$=\lim_{ x \to 0 } \frac{e^x-\sin x-1}{\frac{1}{2}x^2}$
$=\lim_{ x \to 0 } \frac{e^x-\cos x}{x}$
$=\lim_{ x \to 0 } \frac{e^x+\sin x}{1}$
$=e^0+0$
$=1$


2. $\frac{\infty}{\infty}$ 型
> [!important] 特殊方法：抓大头
$x\to +\infty$ 时，$x\ll x^2$，$\ln x\ll x^a\ll a^x$
$x\to +\infty$ 时，$x+x^2\sim x^2$，$x^2+e^x\sim e^x$

> [!example] 例题#1
方法一：洛必达法则
$\lim_{ x \to \infty } \frac{2x^2+3x+4}{5x^2+9}$
$=\lim_{ x \to \infty } \frac{4x+3}{10x}$
$=\lim_{ 4 \to 10 }$
$=\frac{2}{5}$
方法二：抓大头
$\lim_{ x \to \infty } \frac{2x^2+3x+4}{5x^2+9}$
$=\lim_{ x \to \infty } \frac{2x^2}{5x^2}$
$=\frac{2}{5}$


3. $0\cdot \infty$ 型
> [!important] 特殊方法：
化为 $\frac{\infty}{\frac{1}{0}}$ 或  $\frac{0}{\frac{1}{\infty}}$ ，再洛必达法则

> [!example] 例题#1
$\lim_{ x \to 0^+ } x\cdot lnx$
$=\lim_{ x \to 0^+ } \frac{\ln x}{\frac{1}{x}}$
$=\lim_{ x \to 0^+ } \frac{\frac{1}{x}}{-\frac{1}{x^2}}$
$=\lim_{ x \to 0^+ } (-x)$
$=0$


4. $\infty-\infty$ 型
> [!important] 特殊方法
通分->倒代换->洛必达

> [!example] 例题#1
$\lim_{ x \to \infty } \left[ x-x^2\ln\left( 1+\frac{1}{x} \right) \right]$
$=\lim_{ t \to 0 } \left[ \frac{1}{t}-\frac{\ln(1+t)}{t^2} \right]$
$=\lim_{ t \to 0 } \frac{t-\ln(1+t)}{t^2}$
$=\lim_{ t \to 0 } \frac{1-\frac{1}{1+t}}{2t}$
$=\lim_{ t \to 0 } \frac{t}{2t(1+t)}$
$=\lim_{ t \to 0 } \frac{1}{2(1+t)}$
$=\frac{1}{2}$


5. $1^\infty$ 型
> [!important] 特殊方法：万能公式法
$\lim_{} f(x)^{g(x)}=e^{\lim_{} (f(x)-1)\cdot g(x)}$

> [!example] 例题#1
$\lim_{ x \to 0 } \left[ 2-\frac{\ln(1+x)}{x} \right]^\frac{1}{x}$
因 $\lim_{ x \to 0 } \left[ 2-\frac{\ln(1+x)}{x} \right]=1$ ，原式为 $1^\infty$
$=e^{\lim_{ x \to 0 } [2-\frac{\ln(1+x)}{x}-1]\cdot \frac{1}{x}}$
$=e^{\lim_{ x \to 0 } \frac{x-\ln(1+x)}{x^2}}$
$=e^{\lim_{ x \to 0 } \frac{1-\frac{1}{1+x}}{2x}}$
$=e^{\lim_{ x \to 0 } \frac{x}{2x(1+x)}}$
$=e^{\frac{1}{2}}$


6. $0^0$ 型
> [!important] 特殊方法：抬底法
> $\lim_{} f(x)^{g(x)}=e^{\lim_{} g(x)\ln f(x)}$

> [!example] 例题#1
$\lim_{ x \to 0^+ } x^{\sin x}$
$=\lim_{ x \to 0^+ } e^{\sin x\ln x}$
$=e^{\lim_{ x \to 0^+ } x\ln x}$
$=\lim_{ x \to 0^+ } \frac{\ln x}{\frac{1}{x}}$
$=e^{\lim_{ x \to 0^+ } \frac{\frac{1}{x}}{-\frac{1}{x^2}}}$
$=e^{\lim_{ x \to 0^+ }(-x)}$
$=e^0$
$=1$


7. $\infty^0$ 型
> [!important] 特殊方法：抬底法
> $\lim_{} f(x)^{g(x)}=e^{\lim_{} g(x)\ln f(x)}$

> [!example] 例题#1
$\lim_{ x \to 0^+ } \left( \frac{1}{x} \right)^{\tan x}$
$=e^{\lim_{ x \to 0^+ }\tan x\cdot \ln \frac{1}{x}}$
$=e^{-\lim_{ x \to 0^+ }x\cdot \ln x}$
$=e^0$
$=1$


# 数列求极限

> [!important] 夹逼准则：
对于数列 ${x_{n}}$ ， $z_{n}\leq x_{n}\leq y_{n}$ ，且 $\lim_{ n \to \infty } y_{n}=\lim_{ n \to \infty } z_{n}=A$ ，则 $\lim_{ n \to \infty } x_{n}=A$


> [!example] 例题#1
> 设数列 $x_{n}=\frac{1}{n^2+n+1}+\frac{2}{n^2+n+2}+\dots+\frac{n}{n^2+n+n}$ ，求 $\lim_{ n \to \infty } x_{n}$


> [!example] 例题#2
> 设 $a_{1}=6$ ， $a_{n}=\sqrt{ 6+a_{n-1} }$ ， $n=2,3,\dots$ 。
> (1) 证明：$\lim_{ n \to \infty }a_{n}$存在
> (2) 求 $\lim_{ n \to \infty }a_{n}$
