# 函数的连续性

> [!important] 知识储备：
> $\lim_{ x \to x_{0} } f(x)=f(x_{0})$ $\leftrightarrow$ 则 $f(x)$ 在 $x_{0}$ 处连续（左极限等于右极限）


# 函数的间断点

> [!important] 知识储备：
>- 第一类间断点（左右极限均存在，但极限不一定相等）
>1. 可去间断点：$\lim_{ x \to x_{0}^- } f(x)=\lim_{ x \to x_{0}^+ }f(x)\neq f(x_{0})$
>2. 跳跃间断点：$\lim_{ x \to x_{0}^- }f(x)\neq \lim_{ x \to x_{0}^+ }f(x)$
>- 第二类间断点
>1. 无穷间断点：$\lim_{ x \to x_{0}^- }f(x)=\infty$ 或 $\lim_{ x \to x_{0}^+ }f(x)=\infty$
>2. 振荡间断点：$\sin \frac{1}{x},\cos \frac{1}{x}$ 在 $x=0$ 处
