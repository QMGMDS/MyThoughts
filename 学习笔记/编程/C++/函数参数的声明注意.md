
三种不同传参方式：
```c++
// 传值方式 - 会复制整个向量（低效）
void func1(vector<int> nums) {
    // 修改nums不会影响原向量
}
// 传引用方式 - 不复制，高效
void func2(vector<int>& nums) {
    // 修改nums会影响原向量
}
// const引用 - 不复制，且保证不修改原向量
void func3(const vector<int>& nums) {
    // 不能修改nums，只能读取
}
```

拓展 - `vector` ：
C++ 标准模板库（STL）中的一个**动态数组**容器，定义在 `<vector>` 头文件中。`vector<int>` 特指存储整数类型的 `vector`
`vector` 中常用的函数方法：

| 函数      | 用途       |
| ------- | -------- |
| front() | 返回第一个元素  |
| size()  | 返回当前元素个数 |
| back()  | 返回最后一个元素 |
