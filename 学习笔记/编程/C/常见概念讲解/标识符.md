# <font color=008ac9>定义</font>

- 标识符是用户编程时使用的名字

# <font color="008ac9">命名规则</font>

- 标识符由字母(A-Z,a-z)、数字(0-9)、下划线组成
- 标识符的首字符不能是数字，可以是字母或下划线
- 标识符严格区分大小写
- 标识符不能与关键字相同![[屏幕截图 2025-09-26 170229.png]]
