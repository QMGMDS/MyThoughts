
> [!note] 观点
***单例模式*** 的本质是对 **类** 做了特殊化
干了两件事，即：
>- 确保该类只有一个实例
>- 提供该类的全局访问点，让外部只能通过该访问点来访问该类


> [!summary] 概述
***单例模式*** 是一种创建型设计模式，它确保一个类只有一个实例，并提供一个全局访问点来获取这个实例，这个全局访问点可以是静态属性或者方法


**示例：**

```c#
public class Singleton
{
    private static Singleton _instance;
    // 私有构造函数，防止外部实例化
    // 确保只有一个实例
    private Singleton() { } // 该句至关重要（确保一个类只有一个实例）
    
    // 提供全局访问点： Instance 这个静态属性
    public static Singleton Instance
    {
        get
        {
            if (_instance == null)
            {
                _instance = new Singleton();
            }
            return _instance;
        }
    }
    
    public void ShowMessage()
    {
        Console.WriteLine("Singleton Instance Method Called");
    }
}

// 使用
Singleton.Instance.ShowMessage();
```


**重点：**

若没有 `private Singleton() { }` 声明私有构造函数
```c#
public class BadSingleton
{
    private static BadSingleton _instance;
    
    // 没有构造函数，默认是 public 的
    public static BadSingleton Instance
    {
        get
        {
            if (_instance == null)
            {
                _instance = new BadSingleton();
            }
            return _instance;
        }
    }
}

BadSingleton s2 = new BadSingleton();  // 糟糕！可以直接new！
```

有私有构造函数
```c#
public class GoodSingleton
{
    private static GoodSingleton _instance;
    
    private GoodSingleton() 
    {
        Console.WriteLine("Singleton created!");
    }
    
    public static GoodSingleton Instance
    {
        get
        {
            if (_instance == null)
            {
                _instance = new GoodSingleton();
            }
            return _instance;
        }
    }
}

// 使用
GoodSingleton s1 = GoodSingleton.Instance;  // 输出 "Singleton created!"
GoodSingleton s2 = GoodSingleton.Instance;  // 不会再次输出，使用同一个实例
// GoodSingleton s3 = new GoodSingleton();  // 编译错误！
```
