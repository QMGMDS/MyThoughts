
> [!note] 观点
***JSON*** 是一种数据交换的格式，本质是一种语法，通过这个格式，可以将一些特定的数据（JSON对象）通过 ***JSON序列化*** （行为）转化为占用更小的 ***JSON字符串*** （传输格式），通过存储 ***JSON字符串*** 实现存储这些特定的数据


# JSON

> [!summary] 概述
***JSON*** 是一种轻量级的数据交换格式，它基于文本、易于阅读和编写，同时也易于机器解析和生成

**主要特点：**

- 易于阅读和编写  
    采用纯文本格式，结构清晰（类似编程中的对象或字典）

- 独立于语言
    虽然源自 JavaScript，但几乎所有编程语言都支持 JSON（如 Python、Java、C# 等）。

**语法规则：**

- 数据以 *键值对* 形式存在（例如 `"name": "张三"`）。
- 数据之间用逗号分隔。
- 花括号 `{}` 表示对象（无序的键值对集合）。
- 方括号 `[]` 表示 数组（有序的值列表）。
- 键名必须是 **字符串**（用双引号包裹）。
- 值可以是：
    - 字符串（必须用双引号）
    - 数字
    - 布尔值（`true`/`false`）
    - 数组
    - 对象
    - `null`

**下面是符合 JSON 语法规则的数据结构表示：**

```json
{
  "name": "张三",
  "age": 25,
  "isStudent": false,
  "hobbies": ["读书", "跑步"],
  "address": {
    "city": "北京",
    "postalCode": "100000"
  }
}
```



# JSON序列化

> [!summary] 概述
***JSON序列化*** 是将数据对象（如变量、数组、类等）转换为 **JSON字符串** 的过程，以便于存储或网络传输。

**主要特点：**

- *JSON序列化* 的结果是一个字符串

- 支持的数据类型：对象、数组、字符串、数字、布尔值、`null`

- 不支持：函数、循环引用等特定语言特性





# JSON字符串

> [!summary] 概述
***JSON字符串*** 本质是符合 *JSON* 格式规范的文本字符串

**下面是一个JSON字符串：**

```JSON
{"user":{"name":"李四","hobbies":["读书","游泳"],"address":{"city":"北京","postcode":"100000"}}}
```
