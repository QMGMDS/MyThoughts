
# 值类型（Value Types）

值类型的变量直接包含数据，存储在**栈（Stack）** 上
赋值时创建的是数据的副本

包含：
1. **基本数值类型**
    - `int`, `uint`
    - `long`, `ulong`
    - `short`, `ushort`
    - `byte`, `sbyte`
    - `float`, `double`, `decimal`
    - `char`

2. **布尔类型**
    - `bool`

3. **结构体（struct）**
    - 自定义的 `struct`

4. **枚举（enum）**
    - 如 `public enum Color { Red, Green }`



# 引用类型（Reference Types）

引用类型的变量存储的是对数据的**引用（内存地址）**，数据本身存储在**堆（Heap）** 上
赋值时复制的是引用（指向同一对象）

包含：
1. **类（class）**
    - 自定义类
    - `string`
    - `object`

2. **数组（array）**
    - `int[]`, `string[]`
