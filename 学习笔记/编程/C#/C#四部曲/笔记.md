## 变量

14 种基本类型：

- 特殊类型
	- `bool` 布尔型，取值为 `true` 或 `false` 
	- `char` 字符类型，C++中能被当做整型对待，字符对应的数值就是它的 ASCII 码值
	- `string` 字符串类型，没有存储上限
- 整数类型
	- 有符号整型
		- `sbyte` 有符号字节，存储范围：-128 ~ 127
		- `short` 有符号短整型，存储范围：-32768 ~ 32767
		- `int` 有符号整形，存储范围：-21亿多 ~ 21亿多
		- `long` 有符号长整型，存储范围：-9百万兆 ~ 9百万兆
	- 无符号整型
		- `byte` 无符号字节，存储范围：0 ~ 255
		- `ushort` 无符号短整型，存储范围：0 ~ 65535
		- `uint` 无符号整形，存储范围：0 ~ 42亿多
		- `ulong` 无符号长整型，存储范围：0 ~ 18百万兆
- 浮点类型
	- `float` 单精度浮点数，存储范围：存储 7/8 位有效数字
	- `double` 双精度浮点数，存储范围：存储 15/17 位有效数字
	- `decimal` 长双精度浮点数，存储范围：存储 27/28 位有效数字

## 反射

一个运行的程序可以查看本身或者其它程序的元数据的行为就叫做反射
实现反射后：
无需通过在该程序集中 new 出其他程序集的类进而访问该类对象
而是可以直接去访问

为了能实现反射这样的行为
C#准备了：
Type 这个类 即存储其他类的信息的类
Assembly 这个类 即存储其他程序集的类
通过 Type 就可以获得到该类的所有公共成员（包含变量方法构造事件接口属性等等）
也可以访问甚至设置该类的所有实例
通过 Assembly 就可以加载其他程序集中的信息 进而使用 Type 获取该程序集中的类的信息
