
> [!summary] 概述
> ***快速排序*** 是一种基于分治思想来递归解决问题的原地、高效但不稳定的比较排序算法，通过根据基准值不断的分割子数组以达到排序结果

**分治思想：**
	从数组中选取一个数作为**基准**（pivot），遍历该数组，把小于基准的数字放在数组的左边，大于基准的数字放在数组的右边，最终使得：
	- 数组左边所有元素 <= 基准值
	- 数组右边所有元素 >= 基准值

**特点：**
	- 不稳定排序：数组中相等的元素可能会改变相对次序
	- 平均时间复杂度为 O(n log n)
	- 平均空间复杂度为 O(log n)


**代码示例：**

```C++
void quick_sort(int q[], int l, int r)
{
	if (l >= r) return;
	// 这里选取中间索引上的数作为基准（pivot）
	// l+r >> 1 等价于 (l+r) / 2，执行位移指令比除法指令快得多
	int i = l - 1, j = r + 1, x = q [l+r >> 1] ;
	while (i < j)
	{
		do i++; while (q[i] < x) ；
		do j--; while (q[j] > x) ;
		if (i < j） swap(q[i] , q[j]) ;
	}
	quick_sort(q, l, j) ;
	quick_sort(q, j+1, r) ;
}
```

注：示例中以数组或子数组的中间索引位置上的值作为基准，也可以设定范围为头尾索引的随机数作为基准
例如：

```c++
void quick_sort(int q[], int l, int r)
{
	if (l >= r) return;
	int i = l - 1, j = r + 1, x = q [rand() % (r-l+1) + l] ;
	while (i < j)
	{
		do i++; while (q[i] < x) ；
		do j--; while (q[j] > x) ;
		if (i < j） swap(q[i] , q[j]) ;
	}
	quick_sort(q, l, j) ;
	quick_sort(q, j+1, r) ;
}
```

这样能极大降低所选基准恰好为数组所有元素中的最小/大值（此为最坏情况）的概率
